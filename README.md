# Caged Animal Perception Fix Mod
This mod fixes the bug where an awakened animal's perception is fixed at 10 when it comes from cages.

修复从笼子里开出的开智期动物固定拥有10神识的问题。

## Credits
It is originally part of iguana's [iguana_acs_fix](https://github.com/iguanacore/iguana_acs_fix), with slight modification to make it work on mobile.
